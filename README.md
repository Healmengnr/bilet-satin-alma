# HealmeGO - Otobüs Bileti Platformu

PHP ile yazılmış basit otobüs bileti satış sistemi.

## Kurulum

```bash
docker-compose up -d
```

Site: http://localhost:8080

## Güvenlik

- SQL injection koruması
- XSS koruması  
- Path traversal koruması
- CSRF token sistemi
- Şifre hash'leme
- Session güvenliği
